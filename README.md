# WPF_Label (.NET 8.0)

本项目为WPF Label控件的演示程序，涵盖Label的基础用法、Target属性、样式与模板、数据绑定、可访问性等典型场景。
![Uploading image.png…]()

## 主要演示内容
- Label基础用法
- Target属性与访问键
- 样式与模板自定义
- 数据绑定与格式化
- Label与表单控件配合
- 可访问性与表单最佳实践

## 运行方式
1. 使用 Visual Studio 2022 或更高版本打开 WPF_Label.sln（如未生成可自行新建解决方案并添加WPF_Label项目）。
2. 选择 .NET 8.0 SDK。
3. 编译并运行。

## 注意
- 本程序用于对WPF控件的了解和学习
